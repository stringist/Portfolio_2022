Hacked Hogwarts List

This Harry Potter-themed project is from our six-week 'Just JavaScript' theme, in which we learned a great deal about string methods, array methods, working with many moving parts, conditional logic, and generic functions. We received the student data in a JSON file that was riddled with errors such as random capitalization, and the first step on this complex project was to clean the data so we had a properly formatted and capitalized set of data to work with.

See the project here ➔

The user can sort students by first name or last name, house, blood status, and whether they are prefects or members of the 'inquisitorial squad'. They can also expel students, appoint and un-appoint prefects, and appoint inquisitors with the requisite blood status. There is also a built-in 'hack the system' function which inserts the user into the student list and makes them unexpellable, and affects some of the other functions, for example randomizing the blood status algorithm in order to sabotage the inquisitorial squad system.

I wanted to create something that reflected the visual universe of the Harry Potter series, and to that end I tried to make something that looked somewhat like it was hand-written on parchment. I created a series of icons to further this aesthetic.

One of the many challenges in coding this project was to be able to check the student names imported from one JSON file against a list of pure- and half-blood family names listed in a separate JSON file.
